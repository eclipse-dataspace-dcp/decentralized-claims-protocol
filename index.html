Eclipse Decentralized Claims Protocol

Dataspaces require the ability to communicate participant identities and credentials to secure data access. This specification defines a set of protocols for asserting participant identities, issuing verifiable credentials, and presenting verifiable credentials using a decentralized architecture for verification and trust.

This specification is a work-in-progress affiliated with the Eclipse Dataspace Working Group that has not yet reached draft status. If you have a question or would like to comment on an aspect of this specification, please open a discussion. If you have found an error, please file an issue.
